Responsive Navigation Bar

This responsive navigation bar is built with Bootstrap 5. It features:

  • Responsive design that adapts to all screen sizes
  • Hamburger menu toggle on mobile devices
  • Dropdown menus for nested navigation items
  • Smooth hover effects and active state indicators
  • Call-to-action button for important actions
  • Modern gradient background with subtle shadows
  • Mobile-first approach with optimized touch targets

Resize your browser window to see how the navigation bar transforms. On larger screens, all menu items are visible. On smaller screens (mobile devices), the menu collapses into a hamburger icon that expands when clicked.

The navigation includes a dropdown menu under "Services" to demonstrate nested navigation items. The "Get in Touch" button remains visible as a prominent call-to-action across all device sizes.
